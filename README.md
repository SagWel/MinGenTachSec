# MinGenTachSec

Mini gestionnaire de tâches sécurisé.

Pages prévues :

- Page acceuil (not logged) : Accés à login et à register.

- Page Login : connexion (email/password), création de token.

- Page register : verif email -> BDD, verif password -> sécuritée, création token.

- Page acceuil (logged) : Accés liste des taches, gestions des taches.

Front : React-js-tailwind

Back : NodeJs - MySQL
